Тестирование API Яндекс Прилавок
* Чек-лист для проверки функционала наборов, корзины и расчета стоимости доставки.
* Покрыл позитивные и негативные сценарии, выполнив проверки через Postman.
По результатам тестирования оформил 40 баг-репортов и выявил 2 блокирующих дефекта, которые могли негативно повлиять на работу приложения.
